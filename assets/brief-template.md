目的地：

出行人数：
住宿区域，如果已定：
旅行风格：美食 / 亲子 / citywalk / 拍照 / 博物馆 / 轻松 / 特种兵 / 预算优先
必须去的地点：
不想去/忌口/体力限制：
预算：
地图偏好：高德优先
输出要求：生成可部署、手机可打开的单文件 travel-plan.html

小红书来源要求：
只使用小红书真实笔记。
必须先打开浏览器到小红书，让我扫码登录。
登录确认前不要开始搜索或生成攻略。
不要让我提供账号密码、Cookie 或验证码。
